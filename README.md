---

## 📌 About the Project

This project is a front-end BMI calculator developed as a practical assignment for the **CET TPSI** course at **ATEC — Training Academy**.

The page calculates the Body Mass Index based on the user's weight and height, displays the result with the corresponding WHO classification, and handles invalid inputs with visible error feedback — all without reloading the page.

---

## ✨ Features

| Feature | Description |
|---|---|
| ⚖️ BMI Calculation | Calculates BMI from weight (kg) and height (m) using `bmi.js` functions |
| 🏷️ WHO Classification | Displays the result category according to WHO standards |
| ❌ Input Validation | Shows an error message for invalid or missing values |
| 📊 Reference Table | Collapsible table with all WHO BMI categories and ranges |
| 🎨 Bootstrap 5 Styling | Responsive layout styled with Bootstrap 5 components |

---

## 🛠️ Technologies and Concepts Applied

- **HTML5** — semantic page structure
- **CSS3** — custom styling combined with Bootstrap utilities
- **JavaScript**
  - DOM manipulation
  - Event listener on button click
  - Dynamic result display without page reload
  - Input validation with user feedback
- **Bootstrap 5** — layout, form components, alerts, and table styling
